# Amazon-Sales-Analysis-SQL

## Overview

This project is an SQL-based database management and analysis system designed for an Amazon-style e-commerce platform. The project focuses on creating and managing relational database tables, applying constraints, establishing relationships between tables, and performing business-oriented SQL queries for customer, product, order, supplier, and review analysis.

The project demonstrates practical SQL skills including database creation, table modification, primary and foreign key implementation, data manipulation, and analytical query writing using MySQL.

## Features

* Database creation and table management
* Primary key and foreign key relationships
* Data validation using constraints
* Insert, update, and delete operations
* Customer and product analysis
* Revenue and sales calculations
* Product category analysis
* Supplier and inventory tracking
* Review and rating management
* Business reporting queries

The purpose of this project is to simulate a real-world e-commerce database system and perform meaningful business analysis using SQL queries. It helps in understanding how relational databases are used in online shopping platforms like Amazon.
